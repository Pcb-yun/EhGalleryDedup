# Eh下载文件查重工具

## 功能介绍

本工具用于扫描指定项目文件夹中的所有子文件夹，识别并归类高度相似的图像文件夹，以便用户进行内容比对和重复数据管理。

## 主要功能

- 扫描指定根目录下所有符合以下命名格式的子文件夹：
  - `xxxxxxx-xxxx` 格式（纯数字序列号-名称）
  - `xxxxxxx-[xxx] xxxxx` 格式（带扩展信息的序列号-名称）
- 分析各子文件夹中的图像内容（jpg、png、webp、gif等图像格式）
- 识别名称相同或高度相似但序列号不同的子文件夹
- 识别内容高度重复但不同版本的子文件夹
- 提供用户界面展示相似文件夹组，支持用户对每组文件夹内容进行可视化比对
- 提供删除重复文件夹的功能，需包含确认机制防止误操作

## 技术实现

- **编程语言**：Python 3
- **GUI框架**：PyQt5
- **图像处理**：Pillow、OpenCV
- **数据处理**：NumPy、SciPy
- **相似度算法**：感知哈希(Perceptual Hashing)
- **并发处理**：多线程（文件夹扫描）、多进程（图像哈希计算）
- **性能优化**：图片缓存机制、延迟加载、批量处理

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行主程序：
   ```bash
   python main.py
   ```

2. 在界面中点击"选择文件夹"按钮，选择要扫描的根目录

3. 点击"开始扫描"按钮，程序会自动扫描文件夹并计算相似度
   - 扫描过程中可以点击"暂停"按钮暂停扫描
   - 点击"停止"按钮可以终止当前扫描

4. 扫描完成后，左侧会显示相似文件夹组，点击组可以查看详细信息
   - 支持按名称相似度或标签进行分组查看

5. 在右侧文件夹列表中选择文件夹，可以在下方预览其中的图像
   - 双击预览图片可以打开大图查看
   - 窗口大小改变时图片会自动重新排列以适应窗口

6. 选择要删除的文件夹，点击"删除选中文件夹"按钮，确认后即可删除

## 性能说明

- **扫描速度**：使用多线程技术高效处理包含数百个子文件夹和数千张图像的项目
- **内存管理**：优化内存使用，通过图片缓存和延迟加载机制避免处理大量图像时出现内存溢出
- **算法效率**：使用感知哈希算法计算图像相似度，平衡速度与识别精度
- **并发优化**：
  - 文件夹扫描使用多线程（IO密集型任务）
  - 图像哈希计算使用多进程（CPU密集型任务）
  - 自动根据CPU核心数调整线程/进程数量
- **缓存机制**：图片缩略图缓存和哈希值缓存，减少重复计算
- **响应性能**：延迟加载和按需加载确保界面流畅响应

## 注意事项

- 删除操作前会有确认步骤，防止误操作
- 程序只处理符合以下命名格式的子文件夹：
  - `xxxxxxx-xxxx` 格式（纯数字序列号-名称）
  - `xxxxxxx-[xxx] xxxxx` 格式（带扩展信息的序列号-名称）
- 支持的图像格式：jpg、jpeg、png、webp、gif
- 快速滚动预览时，图片会按需加载以确保界面流畅
- 双击预览图片可以打开独立的大图查看窗口
- 调整窗口大小时，图片预览会自动重新排列以适应新的窗口尺寸
