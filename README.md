# EhGalleryDedup

**Eh画廊查重** - EhViewer下载画廊文件夹重复检测工具

## 简介

用于扫描、识别和管理 EhViewer 下载的重复画廊文件夹，通过分析文件夹名称和图像内容检测相似或重复的画廊。

## 功能特性

- 自动识别标准命名格式的画廊文件夹
- 名称相似度 + 内容相似度双重检测
- 树形视图分组展示，缩略图预览
- 全屏图片查看器（滚轮缩放、拖拽平移）
- 实时搜索过滤，安全删除到回收站

## 安装

### 快速使用（推荐）

直接下载 Releases 中的 `Eh画廊查重.exe`，双击运行即可，无需安装 Python。

### 从源码运行

**环境要求**：Python 3.8+ / Windows

```bash
pip install -r requirements.txt
python main.py
```

## 使用方法

1. 点击"打开文件夹"选择画廊根目录
2. 点击"开始扫描"分析文件夹
3. 查看分组结果，预览图片内容
4. 选择重复文件夹删除

### 菜单选项

| 菜单 | 功能 |
|------|------|
| 相似度阈值 | 设置检测阈值（50%-90%） |
| 缩略图大小 | 设置预览大小 |
| 分类方式 | 按名称或按标签分组 |
| 自动扫描 | 选择目录后自动开始 |
| 排序方式 | 按修改时间排序 |

### 快捷键（图片浏览器中）

| 按键 | 功能 |
|-----|------|
| Esc | 关闭图片查看器 |
| R | 重置图片适应窗口 |

## 限制说明

- 仅处理符合 EhViewer 命名格式的文件夹（`xxxxxxx-xxxxx` 或 `xxxxxxx-[标签] xxxxx`）
- 支持图像格式：JPG、JPEG、PNG、WebP、GIF
